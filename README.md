## Descrição

API desenvolvido em NestJs, possui integração com RabbitMQ usando Docker.

## Rodar o projeto

Executar ```docker-compose up``` - Levanta todos os containers, inclusive o RabbitMQ.

## Principais tecnologias utilizadas

NestJs
RabbitMQ
Docker
Dockerize
JWT
Prisma
Postgres
